## Export Finished Project
1. Go to `File>Export>Add to Render Queue>`
2. Specify your export location. By default this will say `Not yet specified` in blue text.
    * Render Settings: `Best Settings` Don't need to adjust for now
    * Output Module: `H.264 - Match Render Settings` Don't need to adjust for now
    * Log: `Errors Only` Don't need to adjust for now
    * Output to: `Google Drive/My Drive/Digital Design/YYYY-MM Project Name`
3. Click Render. The export script will run until finished. If your sound is enabled, a happy chime will play
